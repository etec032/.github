# Etec Prof. Dr. Antônio Eufrásio de Toledo · Organização de TI

Bem-vindo à organização **etec032**!  
Nosso principal objetivo é reunir, organizar e facilitar o acesso aos repositórios desenvolvidos pela equipe de TI da escola **Etec Prof. Dr. Antônio Eufrásio de Toledo**.

## Sobre a Organização

Esta organização foi criada para centralizar os projetos de tecnologia desenvolvidos para apoiar alunos, professores e funcionários da Etec. Aqui você encontrará sistemas internos, ferramentas web e integrações que colaboram com o dia a dia da escola.

## Como Contribuir

1. Consulte os repositórios e leia os respectivos READMEs para entender cada projeto.
